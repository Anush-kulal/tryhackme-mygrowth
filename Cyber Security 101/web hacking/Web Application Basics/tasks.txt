##url(uniform resource locator)
*scheme: http, https
*user: username:password
*domain: website name
*port: port number http:80, https:443
*path: path to the resource
*query parameters: key-value pairs that are sent to the server
*fragment: fragment identifier

##Http request: Request line
*Method
*Path
*HTTP version

#Http request methods
*GET: Used to request data from a specified resource
*POST: Used to send data to a server to create or update a resource
*PUT: Used to update a resource
*DELETE: Used to delete a resource
*HEAD: Used to request a representation of a specific resource
*OPTIONS: Used to request information about the communication options for the target resource
*TRACE: Used to request a trace of the request
*CONNECT: Used to establish a tunnel to the target resource

#Http version
*HTTP/1.1
*HTTP/2
*HTTP/3
