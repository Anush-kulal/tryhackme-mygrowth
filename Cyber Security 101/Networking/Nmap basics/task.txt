###Port scanning:who is listioning
#Scanning TCP ports 

Connect scan:
*3 way handshaking
command : nmap -sT [IP_ADDRESS]

syn scan:
*Only send SYN packet
*No 3 way handshaking
*Fastest scan
command : nmap -sS [IP_ADDRESS]

#Scanning UDP ports
*No 3 way handshaking
*Fastest scan
command : nmap -sU [IP_ADDRESS]

#Limiting the target ports
*by default nmap scans 1000 ports
*we can limit the ports using -p by range
>nmap -p 1-100 [IP_ADDRESS]

*nmap -F [IP_ADDRESS]
*Fast scan
*Scan only 100 ports

###Version Detection: Extract More Information

##OS Detection
-O guess the target device os

>nmap -sS -O [IP_ADDRESS]

##Service & Version Detection
-sV detect the version of running services

>nmap -sS -sV [IP_ADDRESS]

*It detect the SSh server are running on the device it detect the version.

-A  os detection,version detection,version detection,and traceroute
